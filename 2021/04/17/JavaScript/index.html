JavaScript

  1. JavaScript
  • 高阶程式语言
  • 直译式程式语言 (脚本式程式语言)
  • 动态型别程式语言 (Dynamic typed language)
  • 在 Browser 中执行的程式语言
  • 用来控制 HTML 中的 Elements
  • 大小写视为不同字元
  1. JavaScript ≠ Java
JavaScript Java
类型 直译式程式语言 编译式程式语言
应用范围 Web Client-Server, Mobile, Web

  1. 使用方法

    • 引用 JavaScript 档案
    <script src="JavaScript档案路径"></script>
    • 使用 HTML 标籤
    <script></script>
    • 例：
    <html>
　<head>
     <script src="JavaScript档案路径"></script>
     <script>
        /* JavaScript 定义区 */
     </script>
　</head>
  <body>
      <div></div>
      <script>
          /* JavaScript 定义区 */
      </script>
  </body>
</html>

  2. 开发人员常用 debug 工具

    • 搭配开发人员工具
    console.log(obj);
    • 中断执行
    alert(msg);

  3. 执行结果受摆放位置影响

    • 直译式程式语言
    • 读到哪，执行到哪
    • 位置影响执行结果

  4. 命名规则

    • 驼峰型
    • 变数/函数首字小写，以大小写区隔不同单字
    • 范例
      • timer
      • loginForm
      • getElement
      • setTitle

变数(Variable)

  1. 变数宣告

    var variable;
var variable = value;
var variable1, variable2, variable3;
var variable1 = value1, variable2 = value2, variable3 = value3;
    • 开发规则
      • 变数的名称命名，应根据值的内容为有意义的名词。
      • JS中应明确宣告每个变数。

  2. 动态型态语言 (Dynamic typed language)

    • 不需事先宣告资料型态
    • 根据变数内的值决定变数型态

  3. 资料型态

    • 简单

      • 未定义 (Undefined, default)
      • 空值 (Null)

    • 基本

      • 布林 (Boolean)
      • 数值 (Number)
      • 字串 (String)

    • 复合

      • 阵列 (Array)
      • 物件 (Object)

    • 特殊

      • 函数 (Function)

        1. 查询资料型态
        • typeof(variable)

  4. 布林 (Boolean)

    • 值
      • true
      • false
    • 函数 (function)
      • Boolean(variable) 强制转为布林型态
    • 方法 (method)
      • variable.toString() 转为字串型态

  5. 数值

    • NaN (Not a Number)
      • 运算错误
        • 无法转成数值的运算、除以0
    • 近似值
      • 浮点数中的整数可能只是近似值
    • 函数 (function)
      • isNaN(variable)判断是否为 NaN
      • Number(variable)强制转为数值型态
      • parseInt(variable)强制转整数
      • parseFloat(variable)强制转浮点数
        1. 方法 (method)
      • variable.toFixed(n)取小数到第 n 位
      • variable.toString()转换为字串

  6. 字串 (String)

    • 单双引号
    • 字串连接: +
    • 函数 (function)
      • String(variable)强制转为字串
    • 属性 (attribute)
      • variable.length字串长度
    • 方法 (method)
      • variable.charAt(n)取得第 n 个字元 (从 0 开始)

  7. 阵列 (Array)

    • 宣告
    var variableArray = new Array();
variableArray[0] = value0;
variableArray[1] = value1;
variableArray[2] = value2;
    var variableArray = new Array(value0, value1, value2);
    var variableArray =[];
variableArray[0] = value0 ;
variableArray[1] = value1 ;
variableArray[2] = value2 ;
    var variableArray = [value0, value1, value2];
  • 值可为任意型态
  • 属性 (attribute)
    • variableArray.length取得阵列长度
  1. 物件 (Object)

  • 属性 (attribute)

    • 属性值可为任意型态

  • 方法 (method)

    var variableObject = {
 'attribute1 ': value1,
 'attribute2 ': value2,
 'method1 ': function(){
     //do something
  }
};
variableObject.attribute3 = value3 ;		//指定属性值
variableObject.attribute1 ;			//取属性值
variableObject.method1();//使用方法

  1. 函数 (Function)

    • 也称为函式，皆指 Function
    • 函数内的变数生命週期只在函数执行期间
    • 目的
      • 不立即执行，等呼叫时才执行
      • 重复使用
    • 类型
      • 具名函数
        • 定义时，即给予函数名称
    function name (argument1, argument2, …, argumentN){
 //do something
}

  • 匿名函数

    • 定义时，没有给予函数名称，而是使用变数储存
    var name = function(argument1, argument2, …, argumentN){
 //do something
}

  • 立即函数

    • 立即执行
    • 常用于只执行一次的程式码
    • 变数生命週期只在括号内
    • 减少全域变数的产生
    (function(argument1, argument2, …, argumentN){
  //do something
})(variable1, variable2, …, variableN);

  • 开发规则

    • 函数名称的命名为『动词』+『名词』，应根据函数的执行内容给予有意义的动词+名词。
    • 函数宣告时需在函数上方加上函数注解，注解应包含函数说明、引数内容 (引数型态、引数英文名称、引数说明)、回传值内容 (回传值型态、回传值说明)
  • 函数注解可加快开发速度、增加程式码的可读性、易于维护

运算子 (Operator)

  1. 算术运算子 (Arithmetic Operator)
    • 变数运算
      • - 取反
      • a+b a加b
      • a-b a减b
      • a*b a乘b
      • a/b a除以b
      • a%b a对b取馀
      • ++a 先对a+1再返回a
      • a++ 先返回a再对a+1
      • –a 先对a-1再返回a
      • a– 先返回a再对a-1
  2. 位元运算子 (Bitwise Operator)
    • 对整数的位元进行运算
      • a & b 交集 (And)
      • a | b 联集 (Or)
      • a ^ b 互斥 (Xor)
      • ~a 补数 (Not)
      • a << b 左移
      • a >> b 带正负号右移
      • a >>> b 补0右移
  3. 指派运算子 (Assignment Operator)
    • 将右边的运算结果指派给左边
      • a = b a = b
      • a += b a = a + b
      • a -= b a = a - b
      • a = b a = a b
      • a /= b a = a / b
      • a %= b a = a % b
  4. 比较运算子 (Comparison Operator)
    • 两数比较(返回值为true/false)
      • a == b 相等
      • a === b 完全相等
      • a != b 不相等
      • a !== b 完全不相等
      • a < b 小于
      • a <= b 小于等于
      • a > b 大于
      • a >= b 大于等于
  5. 逻辑运算子 (Logical Operator)
    • 返回值为true/false
      • a && b 且
      • a || b 或
      • !(a && b) 非
      • !(a || b) 非
  6. 字串运算子 (String Operator)
    • + 字串相加

语法

  1. If-else

    if(condition){
    //条件成立时
}
    if(condition){
    //条件成立时
}else{
   //条件不成立时
}

  • 开发规则：if-else 的 { } 严禁省略

    if(condition1){
    //条件1成立时
}else if(condition2){
   //条件2成立时
}else{
   //条件皆不成立时
}

  • 开发规则：if-else 的 { } 严禁省略

  • If-else 指派缩写(三元运算符)

    var variable = (condition) ? value1 : value2 ;

  • 符合条件时 variable = value1

  • 不符合条件时 variable = value2

  • 原完整写法

    var variable;
if(condition){
    variable = value1 ;
}else{
    variable = value2 ;
}

  1. Switch-case

    switch(variable){
    case value1 :
        //等于 value1 时
        break;
    case value2 :
        //等于 value2 时
        break;
    case value3 :
        //等于 value3 时
        break;
    default:
        //条件皆不成立时
}

  2. For

     for(variableStartValue ; condition ; step){
    //迴圈内容
}

  3. For-in (for-each)

    • 不须指定起始值、条件、增减值
    • 将所有索引值暂存到 tempVariable
    for(tempVariable in variable){
    //迴圈内容
}

  4. While

    while(condition){
    //迴圈内容
}

  5. Do-while

    • 先做一次迴圈内容再进行判断
    do{
    //迴圈内容
}while(condition)

  6. break 强制跳出迴圈

    • 通常搭配 if 条件使用
    • 任何迴圈皆可使用
    for(variableStartValue ; condition ; step){
    //迴圈内容 
    if(condition){
        break;
    }
}

  7. continue 强制跳至下一次迭代

    • 通常搭配 if 条件使用
    • 任何迴圈皆可使用
    for(variableStartValue ; condition ; step){
    //迴圈内容 
    if(condition){
        continue;
    }
}

  8. Try-catch 错误监测

    • throw 抛出错误讯息并中断执行
    try{
    //错误监测区块
    if(condition){
        throw errorValue ;
    }
}catch(errorVariable){
    //错误发生时的执行内容
}

Document Object Model（DOM）

  1. 简介
    • 将 HTML 中的各标籤 (tag) 转以树状物件的方式呈现

  • 提供 HTML 文件浏览、处理和维护阶层架构的功能
    • 浏览 (Navigator) : 走访各节点
    • 参考 (Reference) : 存取节点
  • 每个节点都是物件

  1. 取得元素节点的方式

    • 元素节点 (Element)
    • id
    document.getElementById(elementId)

  • 节点阵列 (NodeList or HTMLCollection)

    • tag

      document.getElementsByTagName(tagName)

    • name

      document.getElementsByName(celementName)

    • class

      document.getElementsByClassName(className)

  1. 取得元素节点的方式 - HTML5 Selector

    • 元素节点 (Element)
    document.querySelector(selector)
    • 节点阵列 (NodeList or HTMLCollection)
    document.querySelectorAll(selector)
    • HTML5 Selector 与 CSS3 Selector 无太大差异

  2. 元素转物件

    • html
    <div class="className ">innerHTML</div>

    • JavaScript

      • 取得 attribute value
    element.attribute
    • 改变 attribute value
    element.attribute = attributeValue ;

  3. 保留字

    • innerHTML 标籤内容
    • className class 属性

  4. 改变 CSS 样式

    • 改变 style 属性值
      • CSS属性名称有「- 」的，到 JavaScript 要改为大小写模式,例如：font-weight 在 JavaScript 中要改为 fontWeight
      • 不建议使用
    • 定义 CSS 样式后，改变 class

  5. 事件 (Event)

    • 事件为 Element 的其中一个属性 (attribute)
    • 事件需搭配监听器 (Listener, 即函数 Function) 使用
    • 保留引数 event
    • event
      • event.preventDefault() 取消事件的预设动作
      • event.stopPropagation() 停止事件传播

  6. 常用事件

Event 触发时机 常用于
onclick 当滑鼠点击该 Element 时 button
ondbclick 当滑鼠双击该 Element 时
onload 当该 Element 载入完成时 body, img
onchange 当该 Element 改变时 select
onfocus 当该 Element 获得焦点时 input
onblur 当该 Element 失去焦点时 input
onsubmit 当表单被送出时 form
  1. 常用滑鼠事件 (Mouse Event)
Event 触发时机
onmouseover 当滑鼠进入该 Element 时
onmouseout 当滑鼠离开该 Element 时
onmousedown 当滑鼠在该 Element 按下时
onmouseup 当滑鼠在该 Element 弹起时
onmousemove 当滑鼠在该 Element 移动时
  1. 常用键盘事件 (Keyborad Event)
Event 触发时机
onkeydown 当键盘在该 Element 按下时
onkeypress 当键盘在该 Element 按下时
onkeyup 当键盘在该 Element 弹起时
  1. 拖曳事件 (Drag & Drop Event, HTML5)
Event 触发时机
ontouchstart 当该 Element 被触摸时
ontouchmove 当该 Element 被拖曳时
ontouchend 当该 Element 结束触摸时
ontouchcancel 当触摸由系统取消时

动态操作

  1. 动态与静态的区别

    • 静态操作
      • Element 建立在 HTML 档案中
    • 动态操作
      • 透过 JavaScript 操作 Element (新增、修改、删除)

  2. 建立 Element

    1. 建立 Element
    2. 把新建立的 Element 加入到某个节点
    var newElement = document.createElement(tagName);
parentElement.appendChild(newElement);

  3. 删除 Element

    parentElement.removeChild(newElement);

  4. 增加事件监听器

    element.addEventListener(eventType, listener, useCapture)
    • eventType: 事件类型
    • listener: 监听器
    • useCapture: 事件执行顺序
      • true: 由父节点到子节点
      • false: 由子节点到父节点 (default)
    • 增加事件 ≠ 覆盖

  5. 节点相关属性 (Attribute)

    • element.firstChild 第一个子节点
    • element.lastChild 最后一个子节点
    • element.childNodes 所有的子节点
    • element.parentNode 父节点
    • element.previousSibling 上一个相邻节点
    • element.nextSibling 下一个相邻节点
    • element.nodeName 节点标籤名称

  6. 节点相关方法 (Method)

    • document.createElement(tagName) 建立HTML节点
    • element.appendChild(newElement) 增加子节点
    • element.insertBefore(newChild, refChild) 插入子节点
    • element.removeChild(child) 删除子节点
    • element.cloneNode(true) 複製节点
      • true 複製子节点
      • false(default) 不複製子节点
    • element.getAttribute(attribute) 取得属性值
    • element.setAttribute(attribute, value) 设定属性值

常用函数

  1. 讯息框

    • alert(message) 提示讯息框
    • confirm(message) 确认讯息框
    • prompt(message, defaultValue) 输入讯息框
    • 中断 JavaScript 的执行，对使用者会造成不良的使用体验

  2. 计时器

    • setInterval(function, delay 循环执行执行 (无限多次)
    • clearInterval(intervalID) 清除循环计时器
    • setTimeout(function, delay) 时间到后执行 (仅一次)
    • clearTimeout(timeoutID) 清除时间到后执行

  3. Cookie

    • 暂存在 Client 的资料
    • 资料量限制 4 KB
    • 每个 HTTP Request 送出时都会把 Cookie 送到 Server
    • 拖慢执行的效能与浪费不必要的网路频宽

  4. Storage

    • 暂存在 Client 的资料

    • 资料量大

    • 类型

      • Local Storage

        • 透过 JavaScript 清空
        • 使用者手动清除 Browser
        window.localStorage[key] = value ;
        • Session Storage
          • Browser 或 tab 关闭时清空
        window.sessionStorage[key] = value ;

JavaScript Library

  1. 简介
    • 基于 JavaScript 开发，提供开发者快速开发的框架
    • 特色
      • 简化 JavaScript 开发 (Easy-to-use)
      • 浏览器整合 (Cross-Browser)
    • 常见
      1. JQuery
      2. Dojo
      3. Ext.js
      4. …
  2. jQuery
    • 理念：write less, do more
    • 全世界 65% 的网页使用，是目前最受欢迎的 Library (维基百科，2017)
    • Feature-rich
      • DOM 操作 (document traversal and manipulation)
      • 事件处理 (event handling)
      • 动画效果 (animation)
      • Ajax
      • 易用的API (easy-to-use API)
    • 官网：https://jquery.com/
